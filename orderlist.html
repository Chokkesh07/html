Order list

  1. <html>:
    This tag represents the root element of an HTML document. It contains all other elements of the document and must be the first tag in the document.
  2. <head>:
    This tag contains information about the HTML document such as the title of the page, links to external files like CSS and JavaScript, and metadata like keywords and descriptions.
  3. <body>:
    This tag represents the main content of the HTML document. It contains all the visible content like text, images, videos, and other media.
  4. <h1> to <h6>:
    These tags are used for headings and subheadings. The number in the tag indicates the level of the heading, with <h1> being the most important and <h6> being the least important.
  5. <p>:
    This tag is used for paragraphs. It represents a block of text with a line break before and after it.
